Generate some overview diagrams using graphviz. 

This depends on \href{https://www.graphviz.org/}{\tt https\+://www.\+graphviz.\+org/} and will generate pdf files. Filenames start with the prefix specified in the config file as \char`\"{}diagram prefix\char`\"{}.

Draw a modularized diagram showing the dynamic rules of the model. 

Each link type gets one panel with two subpanels for establishment and termination events for this link type. Each subpanel shows all angles influencing the event, annotated with the attempt rate and success probability units contributed by this influence type.
