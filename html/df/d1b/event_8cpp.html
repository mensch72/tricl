Detailed Description

Handling of events.

An event is a tuple {ec, e1, rat13, e3} where ec is the event class (EC_EST or EC_TERM for establishment or termination of a relationship, EC_ACT for the occurrence of an act), e1 and e3 are the affected entities, and rat13 is the relationship or action type.

An event type is a tuple (ec, et1, rat13, et3} where et1, et3 are two entity types.

An event's variable data (current attempt rate, success probability units, and next scheduled timepoint) is stored in a separate struct of type event_data.
